    Determines whether or not to display the sidebar based on an array of conditional tags or page templates.

    If any of the is_* conditional tags or is_page_template(template_file) checks return true, the sidebar will NOT be displayed.
    package WordPress
    subpackage MDG_Base
    param

    list of conditional tags (http://codex.wordpress.org/Conditional_Tags)
    param

    list of page templates. These will be checked via is_page_template()
    return True will display the sidebar, False will not
